# Agent 练手项目清单

> 来源：尚硅谷 AI Coding 零基础实战教程

## 选项目三原则

1. **需求能一句话说清**——边界清晰，不会越做越大
2. **有可视化结果**——能看到东西做出来，保持动力
3. **边界不太大**——3 天到 2 周能完成

## 初级（1-7 天）

| 项目 | 周期 | 练什么 |
|------|------|--------|
| 番茄钟 + 任务记录 | 1-2天 | 状态管理、计时器 |
| 个人记账工具 | 3-5天 | CRUD、图表、数据建模 |
| 习惯追踪器 | 3-5天 | 日历视图、连续打卡 |
| Markdown 笔记应用 | 3-5天 | 编辑器、预览、导出 |
| 个人作品集网站 | 3-7天 | 响应式、部署 |

## 中级（1-4 周）

| 项目 | 周期 | 练什么 |
|------|------|--------|
| 团队任务看板 | 1-2周 | 拖拽、权限、协作 |
| 博客/CMS 系统 | 1-2周 | 内容模型、MDX、后台 |
| URL 短链接服务 | 3-7天 | API、数据库、统计 |
| AI 知识库问答 | 2-3周 | RAG、向量检索 |
| 小型电商 MVP | 2-4周 | 全栈业务、订单流程 |

## 建议路径

```
番茄钟（2天）→ 个人记账（5天）→ Markdown 笔记（5天）→ AI 知识库问答（3周）
```

先跑通一个完整的"想法→实现→部署"循环，建立信心，再逐步加大复杂度。
