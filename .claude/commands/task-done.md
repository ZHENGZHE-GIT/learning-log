# 任务完成收尾

请按以下步骤完成本任务的收尾工作：

## 步骤

1. **确认任务日志**
   - 检查 `docs/` 下是否有本次任务的日志文件
   - 如果没有，按模板创建：`YYYY-MM-DD-任务标题-english-slug.md`
   - 确保 frontmatter 包含 date、tags、summary
   - 确保章节包含：背景、完成事项、踩坑记录、关键文件、下一步计划

2. **更新 README.md**
   - 目录表格新增一行（第一行，最新在前）
   - 标签索引中为新标签添加条目，已有标签追加链接

3. **提交到本地仓库**
   - `git add -A`
   - `git commit -m "docs: 任务标题"`
   - 不执行 git push

4. **展示摘要**
   - 列出本次 commit 包含的文件
   - 统计新增/修改行数
   - 提示用户可以 `/commit` 推送
